# Slack Connection Service

This project provides an **automatic connection list**
to Slack via the **Web API**, retrieving both **users**
and **channels** in the workspace.
****

## Run

- Clone the repository

- Create `.env` file with your Slack Bot Token -
you can copy it from the `.env.sample`

- Run with Docker:

- `docker compose up --build`

    This will build the container, run the script,
    and print the connection list.
